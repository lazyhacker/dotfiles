<leader> = spacebar

<leader>gd - (ctrl-]) Jumps to definition by default then use ctrl-t
<leader>d[v|s] - Jump to the function definition in a new vertical|horizontal split window
<leader>gi - Show the function signature in the status line
<leader>gf - Show the full documentation for function
<leader>gz - Shows all types and function declaration with fuzzy search of current file
<leader>gp - Like :GoDecls but searches the directory
<leader>ga - Toggle between foo.go and foo_test.go
<leader>godoc - Bring up the browser window with the right pkg doc
<leader>gr - go-run
<leader>gb - go-build
<leader>gt - go-test
<leader>gc - go-coverage

F2   - Toggle NERDTree
F3   - Show function definition in status line
F4   - Toggle Paste mode
F5   - Toggle line numbers
F6   -
F7   - Toggle Bufexplorer
F8   - Toggle Tagbar
s-F8 - Toggle and move cursor to Tagbar
F9   - Package documentation in vim
s-F9 - Package doc in browser
F10  - (Tmux) Pop-up shell

<leader>wt - Delete trailing spaces
<C-o> - Jump back to previous position
<C-x><C-o> - Have autocomplete pop-up
gq - Selected text to 80 cols.
xp - swap characters
][ - Jump to end of next function
[] - Jump to end of previous function
`` - returns to previous position
{  - Jump to function
}  - Jump to next function
(  - Previous sentence
)  - Next Sentence
<C-e> - Autopair Wrap
<C-o>x - Delete single char from pair
<C-t>, <C-d>, <C-w>, <C-u>  - Indent, Outdent in INSERT mode, Delete the prefix
<leader>nw - open wiki index file
<leader>ns - list and select wiki
